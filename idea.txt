LA IDEA: Detector de Manipulación y Estafas con IA
El problema sin resolver
En LatAm se pierden $6+ mil millones al año en estafas digitales. Pero el problema real es más amplio — la gente no solo es estafada por criminales, sino manipulada por:

Mensajes de inversión (criptos, esquemas ponzi disfrazados)
Contratos abusivos que nadie lee
Mensajes de WhatsApp de "familiares en peligro"
Romances falsos (catfishing)
Empleadores que violan derechos laborales
Arrendadores que incluyen cláusulas ilegales

Nadie tiene dónde pegar ese texto y que alguien confiable le diga: "esto es trampa, aquí está la prueba".

Cómo funciona (simple)
Usuario pega texto / sube imagen / reenvía mensaje
              ↓
IA analiza en 10 segundos:
  → Patrones de manipulación psicológica
  → Red flags legales según el país
  → Tácticas de estafa conocidas
  → Lenguaje coercitivo o urgencia artificial
              ↓
Resultado: "Esto es legítimo / Señales de alerta / Probable estafa"
+ Explicación en lenguaje simple
+ Qué hacer ahora

Por qué crece solo (sin publicidad)
La mecánica viral está dentro del problema mismo:
Alguien detecta una estafa → lo comparte en el grupo de WhatsApp familiar con el resultado → 40 personas ven el producto funcionando → 8 lo prueban → ciclo.
No vendes un software. Vendes la tranquilidad de no quedar como idiota.

Modelo de negocio (sin publicidad)
Freemium puro:

Gratis: 3 análisis al mes
$7/mes: ilimitado + historial + alertas proactivas
$19/mes (empresas): API para que bancos o fintechs integren el detector

El truco del crecimiento B2B: Bancos, cooperativas, y apps financieras en LatAm pagarían para integrarlo como feature de seguridad para sus usuarios. Eso te da volumen sin adquirir usuarios uno a uno.

Stack (sin capital)

Frontend: React (ya sabes)
Backend: FastAPI
IA: Claude API — es el mejor para razonamiento legal/psicológico
Base de datos: Supabase
Deploy: Render free tier
Pagos: Wompi + Stripe

El costo operativo por análisis es de ~$0.01 USD. Con $7/mes por usuario, el margen es brutal.

Expansión natural (sin reinventar)
Una vez que tienes usuarios, el producto evoluciona solo:

Detector de noticias falsas — mismo motor, diferente input
Analizador de contratos laborales — $39/uso, mercado enorme
Modo relaciones — ¿Este mensaje de esta persona tiene señales de manipulación emocional? (viral en redes de mujeres jóvenes)
API para medios de comunicación — periódicos que quieren verificar fuentes
